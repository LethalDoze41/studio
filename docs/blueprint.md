# **App Name**: Culinary Muse

## Core Features:

- User Authentication: Secure user registration and login using Firebase Authentication.
- Ingredient Input: Dual interface for image uploads (Firebase Cloud Storage) and manual text input with ingredient database.
- Preference Selection: UI for selecting cuisine types, dietary restrictions, meal types, and spice levels.
- AI Recipe Generation: Generate personalized recipes using Google Gemini AI based on selected ingredients and user preferences, acting as a tool to incorporate ingredient knowledge.
- Recipe Display: Display generated recipes in a clear, responsive grid with full details view.
- User Profile: User profile page with saved recipes and recipe history, stored in Firestore.
- Firestore Integration: Store and manage user data, recipes, and preferences in Firestore.

## Style Guidelines:

- Primary color: Fresh avocado green (#A3D36E), for an earthy, inviting tone. (#A3D36E)
- Background color: Light avocado green (#F2F6EC), providing a soft, clean backdrop. (#F2F6EC)
- Accent color: Mellow orange (#D3A86E) for key interactive elements, adding warmth. (#D3A86E)
- Body and headline font: 'PT Sans', a clean and modern humanist sans-serif.
- Use modern and minimalist icons from Heroicons or Lucide React, related to food and cooking.
- Responsive layout with a mobile-first approach, ensuring a seamless experience across devices.
- Subtle transitions and micro-interactions to enhance user engagement.
